# HTML

Lenguaje de etiquetas


```html
<etiqueta atributo="valor" atributo2="valor"> Contenido </etiqueta>
```
# CSS
Lista de Estilos en cascada

```html
selector{ propuedad:valor}

h1{color:red}
````

### Selectores Principales
|nombre    |en HTML               | en CSS                   |
|----------|----------------------|--------------------------|
| etiqueta |<h1>                  | h1{ color:red;}          |
| ID       |<h1 id="titulo">      | #titulo{ color:red;}     |
| class    |<h1 class="destacado">| .destacado{ color:red;}  |
